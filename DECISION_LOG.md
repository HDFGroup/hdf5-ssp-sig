# SSP SIG Decision Log

This file records notable decisions made by the SSP SIG. Each entry
should link back to the relevant GitHub issue or pull request.

## Format

- YYYY-MM-DD — **Title of decision**
  - Summary:
  - Issue/PR:
  - Status: (Active / Superseded)
  - Notes:

---

_Example entries:_

- 2025-01-15 — **Adopt minimum audit checklist v1**
  - Summary: First version of SSP minimum audit checklist ratified.
  - Issue/PR: #12
  - Status: Active
  - Notes: To be revisited after first audit cycle.
